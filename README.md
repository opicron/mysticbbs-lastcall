# Mystic BBS lastcaller mod

## a worthy AmiCaller last call competitor ^^

- wide screen compatible
- scrollable lastcallers list
- additional history based on current scrolled view
- show calls, ULs/DLs, UL/DL size, doors, mails, terminal type, resolution
- 100% configurable, optional before and after draw MCI codes
- full color control of user, location and date strings (per character)
- saves all callers for full history
- fully compatible with callers.dat
- optimized file reading to avoid slowdowns
- save years of caller information for future reference
